# Tools Directory

This directory will contain development and debugging utilities.

Planned tools:
- setup-dev-env.sh - Install dependencies
- load-xdp.sh - Quick XDP program loader for testing
- check-kernel.sh - Verify kernel version and BPF features
- gen-vmlinux.sh - Generate vmlinux.h for multiple kernels
- debug-maps.py - Debug BPF map contents
- packet-tracer.py - Trace packet path through XDP

Helper scripts for development, testing, and troubleshooting.
