# Performance Tests Directory

This directory will contain performance benchmarks.

Planned benchmarks:
- bench_throughput.py - MPPS measurements
- bench_latency.py - Forwarding latency
- bench_srv6.py - SRv6 encap/decap performance
- bench_scale.py - Large FIB scale testing

Uses packet generators (TRex, MoonGen, or pktgen) for load testing.
