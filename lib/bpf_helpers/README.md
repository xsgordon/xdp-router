# BPF Helpers Library

This directory will contain reusable BPF helper functions and macros.

Planned headers:
- bpf_helpers.h - Common BPF helper wrappers
- bpf_debug.h - Debugging macros (bpf_printk wrappers)
- bpf_endian.h - Byte order conversion helpers (if needed beyond libbpf)
- bpf_checksum.h - Checksum calculation helpers

Provides utility functions shared across BPF programs.
